# Mdtero Desktop Deferred Preview Archive

This directory is the archived public mirror surface for old Mdtero desktop preview releases.

The Electron GUI source of truth stays in `JonbinC/mdtero` under `apps/desktop`.

## Current posture

- archived preview surface only
- not part of the current extension-and-CLI launch path
- unsigned by default
- not notarized
- no auto-update yet
- kept only as a deferred reference for past preview artifacts and docs

## Archived preview artifacts

The last mirrored preview version is `0.1.0-preview.1`.

- `Mdtero-0.1.0-preview.1-mac-universal.dmg` → `desktop/releases/Mdtero-0.1.0-preview.1-mac-universal.dmg`
- `Mdtero-0.1.0-preview.1-win-x64.exe` → `desktop/releases/Mdtero-0.1.0-preview.1-win-x64.exe`

These files stay archived for reference only. They are not the recommended way to start using Mdtero right now.

## Boundary

- do not move desktop source code here
- do not treat this directory as an active launch surface
- do not describe this surface as a signed or production desktop release
- keep artifact names and mirrored ledger metadata truthful if this archive is touched
